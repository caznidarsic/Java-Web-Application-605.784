Content:

	This project is a simple Java application.
	
	It makes use of Spring Framework. It consists of two Spring Beans, HelloWorld and HelloWorld2. The main application class creates an instance of each bean, then calls each of their getMessage() methods. Then the context containing these beans is closed. Both beans implement various interfaces that have methods that run on initialization and destruction of the beans. These methods simply print messages to the console, giving insight into the lifecycle of the beans.
